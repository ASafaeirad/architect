# Architect script

[Arch GNU/Linux](https://www.archlinux.org/)+[i3wm](https://i3wm.org/) bootstrap scripts.

## Usage

```bash
  $ git clone git@github.com:ASafaeirad/initiator.git && cd initiator
  $ ./init.sh
  $ ./install.sh
  $ ./config.sh
```
